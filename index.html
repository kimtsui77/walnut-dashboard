✍️ 内容公式（怎么拍）

✅ 趣味剧情是高 A3 种草公式

竞品数据验证：亲子/校园剧情 A3 率 15–24%，远超产品讲解型。核桃「翠花烧烤 14.6% / 段子手小孩 21.4% / 青春期儿子 18.4%」为旗舰素材；自然爆款共性 = 强叙事钩子 + 低品牌侵入感 + 家庭代际共鸣。

✅ 学科/具体痛点驱动看后搜

看后搜率 >0.5% 的视频全部聚焦具体痛点（物理电路、选科焦虑等）。繁星量爆视频同样验证：痛点用「场景描摹」呈现优于直接说教，建议内容 =「AI 辅导 + 编程思维 + 具体学科」组合，用痛点标题触发搜索。

💡 原生剧情赛道空白 = 差异化壁垒

猿编程/西瓜/小码王在「原生剧情」方向几乎为零，核桃系列剧情完全无对手。行业热点端同样显示「师生互动反转剧 / 系列化 IP」是当前教育赛道最高热的内容形态——两端结论互证，应持续加密。

💡 结构因子：量爆 vs 非爆的分水岭

繁星 5–6 月拆解（见下方第二部分）：量爆视频在钩子类型、叙事结构、痛点触达方式上与非爆文差异 ≥15pp 的因子即「强相关因子」，新脚本立项先对照因子表打分。

🎯 投放打法（怎么投）

🔑 先验证爆款再扩量

NOBOOK 5月花费 ▲25% 但 ROI ▼84%，根因是无爆款支撑就扩量。打法：小预算测 3–5 条不同脚本 → 找到 ROI>2 的单条 → 再复制放量。

🔑 IP 系列化 + 多层次达人矩阵

固定 IP 带来认知复利（粉丝追更、边际获客成本递减）；同时避免单一头部依赖（NOBOOK 4月 ROI 主要靠一位达人拉升）。矩阵 = 头部爆量 + 腰部稳量 + 尾部测试。

🔑 品牌词露出决定看后搜下限

NOBOOK 看后搜 <0.1% 的根源是品牌词植入弱。要求：脚本前 5 秒、结尾 CTA、字幕中明确出现「核桃编程」，配合搜索词投放形成闭环。

🔑 选题避坑

规避「编程教育」正面叙事（竞品验证 A3 极低）；AI 话题持续加热，聚焦「不卷成绩 / AI 不替代努力」角度；剧情题材优先「代际冲突 + AI」。
